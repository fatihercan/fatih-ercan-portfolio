# Fatih Ercan Portfolio

Premium one-page portfolio site for GitHub Pages.

## Publish on GitHub Pages

1. Create a public GitHub repository named `fatih-ercan-portfolio`.
2. Upload all files inside this folder, not the zip itself.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/root**, then click **Save**.
6. Your site will be available at:
   `https://YOUR-GITHUB-USERNAME.github.io/fatih-ercan-portfolio/`

## Files

- `index.html` — main website
- `styles.css` — premium visual design
- `script.js` — reveal animation and header behavior
- `assets/fatih-ercan.png` — profile photo
- `assets/Fatih_Ercan_CV.pdf` — downloadable CV
